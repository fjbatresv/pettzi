# Código de conducta

## Nuestro compromiso

Quiero que este repositorio sea un espacio respetuoso, seguro y útil para quienes lo visiten, lo estudien o contribuyan mejoras puntuales.

Se espera que todas las interacciones en issues, pull requests y discusiones mantengan un tono profesional, amable y constructivo.

## Comportamiento esperado

Ejemplos de comportamiento que ayudan a crear un buen entorno:

- usar lenguaje respetuoso y claro
- dar feedback técnico y accionable
- asumir buena intención
- aceptar desacuerdos técnicos sin ataques personales
- priorizar el aprendizaje y la colaboración

## Comportamiento inaceptable

No se toleran, entre otros:

- insultos, acoso o ataques personales
- trolling o provocaciones reiteradas
- publicación de información privada o sensible de terceros
- lenguaje discriminatorio o degradante
- spam o contenido no relacionado con el proyecto

## Alcance

Este código de conducta aplica a las interacciones públicas relacionadas con este repositorio, incluyendo issues, pull requests y discusiones asociadas.

## Moderación

El mantenedor puede moderar, editar, cerrar o eliminar contenido que no cumpla con estas expectativas, así como limitar interacciones cuando sea necesario para proteger el espacio.

## Reportes

Si necesitas reportar una conducta inapropiada, utiliza el canal de contacto público del perfil de GitHub del autor o un canal privado disponible.

## Nota

Este documento está inspirado en principios comunes de convivencia de comunidades open source, adaptado al alcance actual de PETTZI como repositorio público de referencia técnica.
