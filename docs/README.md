# Documentation

This repository keeps its technical documentation as versioned files inside `docs/`.

## Structure

- `docs/reference/content/` — product, architecture, API and infrastructure reference docs preserved as static reference sources
- `docs/specs/` — feature and implementation specs

## Notes

- Documentation is maintained directly in the repository.
