# PETTZI Backend Overview

> Project status: PETTZI is now published as a public technical reference. The managed SaaS effort was discontinued after it was deemed not commercially viable.

PETTZI is a serverless platform for pet health and ownership management:
- Manage pets, owners (PRIMARY/SECONDARY), events, reminders.
- Store documents and photos in S3 via presigned URLs.
- Send transactional emails (welcome, reset, reminders) via SES.

Tech highlights:
- Nx monorepo, Node.js 24.x Lambdas.
- AWS CDK v2, DynamoDB single-table, S3, Cognito, SES, EventBridge.
- One HTTP API per bounded context, mapped under a shared custom domain.

Bounded contexts:
- Auth, Pets, Owners, Events, Reminders, Uploads, Catalogs.
- Domain model and utilities shared through Lambda layers and libraries.
